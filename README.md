# Business District International 🏢

This repository contains the source code for the official website of **Business District International**, a Venture Management Architect Firm. 

The platform is built as a modern Single Page Application (SPA) to showcase the firm's 12-year track record in enterprise formulation, economic consulting, and venture management architecture. It provides a seamless, fast, and interactive user experience without page reloads.

## ✨ Features
* **Lightning Fast Navigation:** Utilizes Vue Router for instant, client-side page transitions.
* **Modern UI/UX:** Styled completely with Tailwind CSS for a highly responsive and professional design.
* **Dynamic Content Handling:** Clean separation of data and UI, especially in complex sections like Services and Governance.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.

## 🛠️ Technology Stack
* **Framework:** [Vue.js 3](https://vuejs.org/) (Composition API)
* **Routing:** [Vue Router 4](https://router.vuejs.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Build Tool:** [Vite](https://vitejs.dev/)
* **Icons:** FontAwesome 6
